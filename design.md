# CreditPulse Design System & Landing Page

## Overview
CreditPulse is a premium, fintech-style platform designed to provide MSMEs with AI-driven Financial Health Cards and lenders with alternative data scoring. The design language prioritizes a clean, authoritative, and modern aesthetic.

## Typography
- **Primary Font**: `TT Norms Pro` (Weights: 400, 600)
- **Fallback**: System sans-serif (`ui-sans-serif, system-ui, sans-serif`)
- **Headings**: Use negative letter-spacing (e.g., `-0.04em`, `-0.03em`) for a tight, modern fintech feel.
- **Weights**: `font-medium` (600) is used as the heaviest weight throughout the landing page.

## Color Palette
- **Background**: Soft Off-White `#F5F5F5` (Landing Page Base) and `#FFFFFF` (White Sections).
- **Cards (Dark)**: Deep Purple/Navy `#2B2644`
- **Primary Text**: Black `#000000` and `Black/70` (`rgba(0,0,0,0.7)`) for secondary text.
- **Buttons (Primary)**: Black `#000000` with White text, hover state `#1F2937` (`gray-800`).
- **Accent Icons**: White circular backgrounds with black icons (e.g., ArrowRight).
- **Footer**: Deep Black `#000000` background with White and White/60 text.

## Components
### 1. Navigation
- Transparent overlay on the hero section.
- Clean wordmark with custom interlocking squares logo.

### 2. Hero Section
- Full-height minus navbar.
- High-quality background video loops.
- Continuous brand marquee showcasing data sources (IDBI Bank, GSTN, UPI, etc.).

### 3. Feature Cards (Meet CreditPulse)
- Mixed layout: one large image-backed card spanning two columns, alongside solid `#2B2644` cards.
- Bottom-aligned text with generous padding.

### 4. Detail Features
- A 4-column grid highlighting specific technical merits of the hackathon project: Time Machine Projections, MSME Owner Portal, Compliance, and What-If Simulations.
- Uses large high-contrast icon blocks.

### 5. Interactive Elements
- Seamless marquee animations (CSS keyframes, 22s and 30s linear infinite).
- Buttons utilize `transition-colors duration-200`.
- All videos autoplay, muted, and inline.

## Page Structure
1. **Hero**: High impact value proposition.
2. **Info/Meet CreditPulse**: 3-card layout explaining core data, AI committee, and insights.
3. **Features (Scale & Precision)**: Deep dive into platform specifics.
4. **Backed By**: Tech stack marquee.
5. **Use Cases**: Split screen with video and specific "Lending" use case.
6. **Footer**: Clean footer with copyright and links.
